# Coffee Sales Analysis
This project explores coffee sales performance by analyzing revenue trends, customer ordering behavior and product performance over time and product types.
Data analysis project analyzing coffee sales data using SQL and Power BI visualization.

## Project Structure

This repository contains the following folders:

### /data
Contains raw data files for the coffee sales analysis project.
- `coffee_sales.csv` - Raw coffee sales data

### /sql
Contains SQL queries for data analysis:
- `01_validation.sql` - Data validation queries for checking data quality
- `02_kpis.sql` - KPI calculations and key metrics
 - ## KPIs
  - Revenue: total sales amount (SUM of money)
  - Orders: number of transactions
  - AOV (Average Order Value): Revenue / Orders
    
- `03_product_analysis.sql` - Product-level analysis and performance metrics



### /powerbi
Contains Power BI dashboard files:
- `coffee_sales_dashboard.pbix` - Main Power BI dashboard

### /screenshots
Contains screenshots and visualizations from the Power BI dashboard:
- `dashboard_page1.png` - Dashboard overview

## Technologies Used
- SQL - Data querying and validation, analysis
- Power BI - Data visualization and dashboarding
